1Scope & who operates this

GTA 6 Insider is an independent, creator-operated TikTok channel about GTA 6 news, rumors, and speculation. This policy covers the internal publishing tool the operator built to draft, review, and post their own videos to their own TikTok account via TikTok's official Content Posting API.

It is not a public app, plugin, or service offered to other creators. No one other than the operator authenticates through it, and no TikTok account other than the one the operator owns is ever connected to it.

2Data we collect

The tool collects only what TikTok's API requires to publish a video and to show the operator whose account they're posting to before each post:

3How we use it

  • Authenticating requests to TikTok's Content Posting API on the operator's behalf.
  • Displaying the connected account (username, avatar) and its real available privacy levels before every publish action, so the operator always knows exactly what they're about to post and to whom.
  • Uploading finished videos and tracking whether the post succeeded.

Data is never used to profile viewers, build advertising audiences, or train any model beyond the operator's own script-writing assistant, which only ever sees the video's own narration text — never TikTok account or viewer data.

4Third-party services

Producing a video involves a small number of processing services. None of them receive TikTok account data — only the content being generated:

  • Anthropic (Claude) — writes narration scripts and image prompts from a news headline or topic.
  • Google (Gemini / Veo) — generates scene images and the video hook clip.
  • Pollinations.ai — generates background-track images.

TikTok tokens, usernames, and avatars are never sent to any of these services. They only ever see the creative content of the video being made.

5Storage & retention

Tokens and video data are stored in a local file store and a local SQLite database on the operator's own machine — there is no shared or third-party server. Tokens are kept until the operator disconnects the app or TikTok revokes them; video files and metadata are kept for as long as the operator wants them for their own review history, and can be deleted at any time.

6Your rights & revoking access

Because the only TikTok account involved belongs to the operator themself, exercising these rights is a matter of the operator acting on their own account:

  • Revoke access anytime from TikTok's Settings → Manage apps and devices, or by deleting the local token file.
  • Delete stored data by removing the local database and video files — there is no remote copy.
  • See what's stored by inspecting the local database directly, since nothing is hidden behind a remote service.

7Security

The OAuth callback used to connect the account runs on a short-lived, single-purpose local endpoint, separate from the review dashboard, and is only exposed during the connection step itself. Tokens are stored in a local file that is never committed to source control or transmitted anywhere except to TikTok's own API.

8Changes to this policy

If the tool's data practices change — for example, if a new third-party processing service is added — this page will be updated and the effective date above will change accordingly.
